Roadmap for You — Starting Fresh from 2nd Year

A focused plan from now (3rd sem) through final placement.

Back to Home View Plan
III

3rd Sem (NOW) — Restart & Catch Up

Focus: Build foundation & start coding seriously.

  • DSA: Pick C++ or Java. Learn arrays, strings, recursion, linked list, stack, queue. Target 100+ problems this sem.
  • Web Dev Basics: HTML, CSS, JS. Build 2–3 small projects (portfolio, todo, calculator).
  • College Subjects: Focus on OOP, Discrete Math.

End goal: 100–150 DSA problems; 1 portfolio site + 1 mini JS project on GitHub.

IV

4th Sem — Intermediate DSA & Backend Basics

  • DSA: Trees, Graphs, DP basics. Goal: 250–300 problems by end of sem.
  • Development: Learn React.js and start backend (Node.js+Express or Java Spring). Learn SQL (MySQL/Postgres).
  • Project: Build first full-stack project (Blog/Notes app with auth).

End goal: 1 deployed full-stack project; ~300 DSA problems solved.

V

5th Sem — Resume Projects & Internships

  • DSA: Advanced topics: DP, Tries, Segment Trees. Reach 400–500 problems.
  • Development: Build 2 strong projects (E‑commerce, Chat app). Contribute to open source / hackathons.
  • Career: Start applying for internships (LinkedIn, Internshala, startups).

End goal: Resume with 3 strong projects + internship or OSS contributions.

VI

6th Sem — Placement Prep & Internships/Jobs

  • DSA: Revise everything. Goal: 600+ problems solved.
  • Core CS: Study OS, DBMS, CN, OOP (interview focus).
  • Development: Work on major project (final/major project).
  • Career: Apply for internships and off-campus jobs.

End goal: Placement-ready skills; internship/job secured or interviews lined up.

VII

7th Sem — Placements & Capstone

Focus on mock interviews, coding tests, final capstone project, and online presence. If not placed, prepare backups (GATE, higher studies).

VIII

8th Sem — Final Steps

Focus on final project & exams. Apply as needed. End goal: job offer or solid backup plan.

Daily / Weekly Routine

  • Daily: 2 hrs DSA (1–2 problems), 1 hr Web Dev, 30 min CS core reading.
  • Weekly: Push code to GitHub, write notes, dedicate 1 day to project work.

Motivation

Starting in 3rd sem is fine — consistency beats speed. Follow this plan and you'll be placement-ready by 6th sem.
